매개변수에는 기본형의 경우 값이 복사되어 들어가고 참조형의 경우 주소값이 복사되어 들어간다.
따라서 기본형과 다르게 참조형은 내부에서 값을 변경하면 외부에 있는 값도 같이 변경이 된다.
이 특징을 이용해서 똑같은 메서드를 void로 변경할 수 있다.
```java

int add(int a, int b) {
    return a + b;
}

int[] arr = {0}
void add(int a, int b, int[] arr) {
    arr[0] = a + b;
}
```

static 메서드는 언제 사용?
인스턴스 변수를 사용하지 않는 메서드
모든 인스턴스 변수가 공유하는 메서드
인스턴스를 사용하지 않고 사용할 수 있음
static 메서드에서 인스턴스 메서드를 호출할 수 없음


클래스 멤버는 인스턴스 멤버를 호출할 수 없는데, 그 이유는 클래스 멤버를 호출하는 시점에 인스턴스가 생성되지 않았을 가능성이 있기 때문이다.
